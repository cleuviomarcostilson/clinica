# VisionCare Angola - Clínica Óptica

Site estático para clínica óptica em Angola, desenvolvido com HTML, CSS e JavaScript.

## Estrutura do Projeto

- `index.html` - Página inicial
- `pages/` - Páginas adicionais (serviços, produtos, contactos, marcação)
- `css/` - Folhas de estilo
- `js/` - Scripts JavaScript
- `assets/` - Recursos estáticos (imagens, etc.)

## Como Usar

Abra o arquivo `index.html` diretamente no navegador. O site funciona sem servidor.

## Funcionalidades

- Design responsivo
- Formulário de marcação com redirecionamento para WhatsApp
- Listas de serviços e produtos
- Navegação suave
- Botão flutuante do WhatsApp

## Tecnologias

- HTML5
- CSS3
- JavaScript ES6+
- Font Awesome
- Google Fonts
